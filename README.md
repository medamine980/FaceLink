## Description

An application that connects two people together, they can talk with eachother through video chatting, also they can text and share links though the text chat section.

## Main technology

#### Backend
- Nodejs (http module)
- Ws (WebSocket) module  

#### Frontend
- HTML/CSS/JS
- WebRTC API
- WebSocket API
